{% blocktrans %}OpenSlides 1.6 besitzt eine Schnittstelle für externe Plugins. Plugins für OpenSlides sind in Python geschriebene Django-Applikationen. Sie können durch spezielle Funktionen in OpenSlides integriert werden.{% endblocktrans %}

{% blocktrans %}Eine Anleitung für Entwickler zum Erstellen eigener Plugins gibt es im Wiki.{% endblocktrans %}

{% trans "Verfügbare Plugins" %}


  • CSV Export Plugin

    URL: https://pypi.python.org/pypi/openslides-csv-export
    {% trans 'Lizenz' %}: MIT
    {% trans 'Autoren' %}: Oskar Hahn, Norman Jäckel

    {% blocktrans %}Dieses Plugin bietet einen CSV-Export der Rednerliste. Es soll außerdem als Beispiel dienen zum Schreiben von Plugins für OpenSlides.{% endblocktrans %}

  • VoteCollector Plugin

    URL: https://pypi.python.org/pypi/openslides-votecollector
    {% trans 'Lizenz' %}: MIT
    {% trans 'Autoren' %}: Oskar Hahn, Emanuel Schütze

    {% blocktrans %}Dieses Plugin ermöglicht die Anbindung von OpenSlides an das elektronische Abstimmungssystem VoteCollector.{% endblocktrans %}

  • Topic Voting Plugin

    URL: https://pypi.python.org/pypi/openslides-topicvoting
    {% trans 'Lizenz' %}: MIT
    {% trans 'Autoren' %}: Norman Jäckel

    {% blocktrans %}Dieses Plugin bietet eine strukturierte Abstimmung über Themen unter Verwendung der Sainte-Laguë-Methode.{% endblocktrans %}

  • Protocol Plugin (Beta)

    URL: https://github.com/OpenSlides/openslides-protocol
    {% trans 'Lizenz' %}: MIT
    {% trans 'Autoren' %}: Norman Jäckel, Emanuel Schütze

    {% blocktrans %}Dieses Plugin ermöglicht die Protokollierung von Tagesordnungseinträgen in OpenSlides.{% endblocktrans %}
